Grid Layout

Basic grid layout using Bourbon and Neat.

To be used as an example, learning tool, and for reference.

Using Bourbon and Neat

  • gem install sass
  • gem install bourbon
  • gem install neat
  • cd to project stylesheet directory
  • bourbon install
  • neat install

Import Statements

Used atop SCSS file to import both Bourbon and Neat

  • @import "libraries/bourbon/bourbon.scss";
  • @import "libraries/neat/neat.scss";

Koala

http://koala-app.com

Used to compile SCSS to standard CSS.

Open your project folder in Koala, create a SCSS file. Each time you save Koala will compile the SCSS file into a CSS file of the same name.

SCSS Variables

Declare SCSS variables with a `$`

  $textcolor: #777777;

Then call them in various elements the same way.

  p{
    color: $textcolor;
  }
